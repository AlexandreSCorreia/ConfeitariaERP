# MVP

## Objetivo

Entregar uma versão funcional capaz de controlar o fluxo de pedidos de uma confeiteira que trabalha sob encomenda, desde o cadastro do cliente até o planejamento da produção.

---

## Funcionalidades

### Clientes

- Cadastrar cliente
- Editar cliente
- Listar clientes

### Produtos

- Cadastrar produtos
- Definir sabores
- Definir preço

### Ingredientes

- Cadastrar ingredientes
- Controlar estoque

### Receitas

- Associar ingredientes aos produtos
- Definir quantidade de cada ingrediente

### Pedidos

- Criar pedido
- Editar pedido
- Alterar status
- Informar pagamento

### Produção

- Visualizar pedidos por data
- Visualizar pedidos do dia

### Compras

- Gerar lista de ingredientes faltantes

---

## Fora do escopo

As funcionalidades abaixo não fazem parte do MVP.

- Dashboard financeiro
- Controle de despesas
- Emissão de nota fiscal
- Integração com WhatsApp
- Integração com Instagram
- Aplicativo mobile
- Relatórios avançados
- Controle de entregadores
- Controle de caixa
- Controle de múltiplos usuários

---

## Critério de sucesso

O MVP será considerado concluído quando a confeiteira conseguir:

- Cadastrar clientes.
- Criar pedidos.
- Planejar sua produção.
- Saber quais ingredientes precisa comprar.
- Marcar pedidos como entregues.
